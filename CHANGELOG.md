# Change Log

## v0.1.0 - 2024-10-17

- Updated Angular version and dependencies from 14.1 to 16.2
- MindMap Cluster drag & drop: 
  - can now drag & drop nodes onto cluster to add that node to the cluster. 
  - Or drop nodes onto another node (not in a cluster) to create a new cluster from those two nodes
  - Or drag nodes out of a cluster
